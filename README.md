# 超級偵探訓練營 - AI 圖片生成器

## ⚙️ Vercel 環境變數設定

部署時需要在 Vercel 設定這個環境變數：

| 變數名稱 | 值 |
|---------|---|
| MINIMAX_API_KEY | 你的 MiniMax Token Plan API Key |

**這樣 API Key 不會存在 GitHub 程式碼中！**

## 🚀 部署步驟

1. 把這個專案 import 到 Vercel
2. 在 Vercel 專案設定中加入環境變數：
   - Key: `MINIMAX_API_KEY`
   - Value: 你的 MiniMax Token Plan API Key
3. 完成！

## 🔐 安全機制

- 密碼保護：55666566
- 速率限制：每 IP 每小時 10 次
- 內容過濾：阻擋不當關鍵字

## 📱 使用方式

1. 打開網址
2. 輸入密碼
3. 輸入圖片描述或點擊範例
4. 點擊生成
5. 下載圖片上傳到 Padlet
